BELLCOS BOLIVIA
IMPLEMENTACION DATOS LOGISTICOS

Objetivos.

  • Organizar datos
  • Limpiar datos
  • Relacionar tablas
  • Interpretar datos
  • Generar Dashboard
  • Toma de desiciones




Desarrollo.
El proyecto de datos, fue solicitado den base a la incertidumbre de la situacion actual de los procesos, donde si bien son funcionales, no se tiene indicadores exactos de los resultados, por lo cual imposibilita la toma de desiciones de mejora a largo plazo.
Si bien cuentas con registros de datos para todos los proceso, estos son almacenados en su sistema SAP, pero no son procesados, ni se les realiza mantenimiento.
Nuestro proyecto debe proporcionar una herramienta que automatice la creacion de tablas y Dashboard, con datos limpios y acertados, lo que permita generar reportes, calcular KPI, visualizar la situacion actual y proyectar los resultados a futuro.
Para tener un control constante sobre el proyecto, fue dividivo en etapas, donde se revisaba el avance y el cumplimiento finalizada cada una, entre las cuales tenemos:

  • Extraccion de datos del SAP
  • Creacion de consultas en SQL
  • Insercion de datos en Power Bi
  • Limpieza, organizacion y seleccion de datos
  • Relacionamiento de tablas
  • Correccion de errores
  • Analisis de problematica y soluciones
  • Generacion de dashboards
  • Toma de desiciones basada en datos
Cuando se cumplio con la planificacion de etapas, se presentaron los resultados a gerencia, con ciertas desiciones sugeridas, para ser evaluadas. Se tuvo la retroalimentacion, se realizo correcciones y se acordo realizar mantenimiento continuo a los datos.

Herramientas.
Este proyecto tuvo varios desafios en el uso de una herramienta nueva para la oraganizacion, como el Power Bi, entre otras, como son:

  • Excel: En la interpretacion de los cuadros finales
  • VisualBasic: En la automatizacion de procesos
  • SQL: En la extraccion de datos de fuentes externas
  • PowerBi: Para la manipulacion, seleccion y limpieza de datos
Conclusiones.
Con el proyecto concluido y en uso, se puede notar que en ocasiones no se tiene un conocimiento claro de la situacion actual que atraviesa uno o varios procesos, esto por no tener acceso e interpretar de forma correcta los datos, teniendo datos limpios y oraganizados formamos las bases para una buena toma de desiciones.

Duracion: 2 Meses
Estado: Concluido
